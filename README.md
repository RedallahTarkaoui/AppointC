# AppointC

AppointC is a console-based appointment management system written in C.

## Features
- Linked list (dynamic memory)
- Sorted insertion by date
- File persistence using rdv.txt
- Search by ID, name, or date
- Modify and delete operations

## How to Run

Compile:
gcc appointc.c -o appointc

Run:
./appointc

To test:
Rename rdv_sample.txt to rdv.txt
